How to download!

These steps require that you have java edition and you have installed the base game, forge, and java.

1

Download

download the OhClient jar file
2

Install

Move the jar file to your mod folder, you can use the official launchers directory button (folder icon) to go to the mod folder (under versions)
3

Running

Once OhClient is in your 1.8 forge mod folder you can start the client.
